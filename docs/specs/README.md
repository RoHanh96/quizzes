# Specs tính năng (quizzes)

Mỗi tính năng mới hoặc thay đổi lớn nên có **một file spec** tại đây.

- Copy `docs/specs/_template.md` → đặt tên mới (ví dụ `001-multiple-choice-export.md`).
- Cập nhật checklist tiến độ trong file đó khi làm việc.
- Trước khi coi xong: chạy `npm run verify` (và `npm test` / e2e khi đã có trong `package.json`), ghi lại trong spec mục **Verify đã chạy**.

Quy ước chi tiết: `.cursor/rules/quiz-delivery-and-docs.mdc`.
